兼职人物

人物在升级时可以选择新的职业，成为兼职人物。不同职业的能力可以累积。兼职可以增加人物的广度，但牺牲了单一职业的深度。

职业与等级特色

一般而言，兼职人物的能力是所有职业能力的总合。

等级：兼职人物的"人物等级"是他所有职业的总等级。人物等级用来决定可否获得新专长或提升属性值，请见22页表3-2：经验值与升级影响。

"职业等级"则是人物在不同职业上的等级。对未兼职人物而言，人物等级和职业等级是一样的。

生命值：兼职人物在职业等级提升时从各职业分别获得生命值，加总成为最大生命值。例如半身人莉达原本是4级游荡者，接下来选择提升两级法师职业等级。身为4级游荡者/2级法师，她的总生命值为：6 + 1d6 + 1d6 + 1d6 + 1d4 + 1d4。

基本攻击加值：把不同职业的基本攻击加值相加，就是该人物的基本攻击加值。如果其结果达到+6或更高，则该人物获得多重攻击能力，从22页表3-1：基本豁免加值与基本攻击加值可得知他的额外攻击次数，以及基本攻击加值为多少。举例而言，6级游荡者/4级法师的基本攻击加值为+6(游荡者职业+4，法师职业+2)。基本攻击加值+6使他获得两次攻击的能力，第二次攻击的加值为+1(如表3-1的+6/+1)，尽管原本的基本攻击加值游荡者职业+4，法师职业+2分开来看都没有额外攻击次数。

豁免检定：把每个职业的基本豁免加值相总，就是人物的豁免加值。7级游荡者/4级法师的强韧检定有+3加值(7级游荡者+2，4级法师+1)，反射检定有+6加值(+5，+1)，意志检定有+6加值(+2，+4)。

技能：在计算最高级数时，只要该技能是人物任一兼职职业的本职技能，即视为人物的本职技能(本职技能的最高级数是3+人物等级)。

如果技能不属于任一兼职职业的本职技能，则技能的最高级数为本职技能的一半。

例如，对7级游荡者/4级法师(人物等级11级)来说，任何游荡者或法师技能的最高级数都是14。不属于游荡者或法师的职业技能为跨职技能，其最高级数是7。

职业特性：兼职人物可得到所有兼职职业的职业特性，但也具有所有职业的特殊限制(例外：兼职野蛮人不会因此变成文盲)。有些职业特性会彼此干扰，例如：虽然游荡者擅长轻型盔甲，但是兼职游荡者和法师的人物，在穿着轻型盔甲时，仍然必须计算奥术失效机率。

驱散不死生物特别说明：圣武士与牧师都有此职业特性，如果兼职圣武士和牧师的人物其圣武士职业等级达到4级，他的有效驱散等级=牧师等级+圣武士等级-3。举例而言，5级圣武士/4级牧师的驱散不死生物能力，等于6级牧师。

直觉闪避特别说明：野蛮人与游荡者都有此职业特性，如果兼职野蛮人和游荡者的人物从第二个职业获得第二次，则以精通直觉闪避取代。在决定夹击所需的攻击者游荡者等级时，野蛮人和游荡者的职业等级要相加。例如，2级野蛮人/4级游荡者只能被10级以上的游荡者夹击。

魔宠特别说明：术士与法师都有此职业特性，兼职术士和法师的人物要把术士等级和法师等级相加，来决定魔宠的天生防御、智力和特殊能力。

专长：对兼职人物而言，人物等级(而非个别职业等级)每上升三级，可以获得一个专长(见22页表3-2：经验值与升级影响)。

属性增加：对兼职人物而言，人物等级(而非个别职业等级)每上升四级，可以获得1点属性值(见22页表3-2：经验值与升级影响)。

法术：兼职人物可获得所有兼职职业的法术。例如，兼职巡林客与德鲁伊者，可能两个职业都可以施展"防护元素伤害"，由于此法术的效果是根据施法者职业等级计算，所以当人物准备与施展"防护元素伤害"时，必须决定是根据巡林客职业或德鲁伊职业。

增加第二职业

当单一职业人物升级时，他可以选择在其旧有职业上继续升级，或选择一个新的职业，兼职该职业1级。(即使该人物可以选择不同的职业特性，例如牧师选择不同的领域，人物依旧不能重复选择相同职业的第1级。)DM可以根据该人物的职业、技能、经验与训练，限制其兼职选择。举例而言，人物得找到可以教导他新职业的导师，才能获得新职业。此外，DM可以要求玩者必须事先声明要钻研另一个职业，不时练习该职业技能，如此一来，新职业的取得比较像是靠努力习得，而不是突然改变。

选择兼职的人物会获得新职业的基本攻击加值、基本豁免加值、本职技能、擅长武器与防具、法术、其他职业特性，以及对应生命骰数的生命值。此外，该人物亦获得新职业的升级技能点数(而非该数字x4，在此和人物等级第1级时不同)。

兼职新职业不完全等同于创造一个该职业的新人物。某些人物等级1级时获得的好处(例如4倍的技能点数)，代表了人物在年轻力壮时接受训练，有较多的练习时间。兼职者在选择一个新职业时，无法得获得以下特性：

  • 根据第一个生命骰数全满的生命值。
  • 起始技能点数乘以4倍。
  • 起始装备。
  • 起始金钱。

升级

人物升级时，可以选择在原有职业升级，或是兼职一个新职业。

若兼职人物选择升级原有职业，他会得到该职业一般在升级时所获得的好处：更高的生命值、可能会提升基本攻击加值、防御等级、基本豁免加值，或是获得职业特性、法术和技能点数。

兼职人物在花费技能点数及购买技能级数的算法，都依据所升级的职业来计算(见62页表4-1：每级技能点数和表4-2：技能)。

有关人物等级超过20级(包含兼职人物)的规则，请见《地下城主指南》。

兼职人物的经验值

兼职不是件轻松的事，依兼职人物的职业等级与种族，他可能会受到XP减值。

等级平衡：如果人物各兼职职业的等级都很接近(彼此相差不超过1级)，则他可以平衡各职业的需求，不会受到XP减值。举例而言，4级法师/3级游荡者不会受到XP减值。2级战士/2级法师/3级游荡者也不会受到XP减值。

等级不平衡：如果兼职人物任两个职业等级相差达2级，则会有不良影响。每有一个职业的等级与最高职业等级相差达2级，该人物就受到-20%的XP减值。此XP减值在该人物增加新职业或升级原有职业时，即开始生效。举例而言，4级法师/3级游荡者不会受到XP减值，但当他的法师职业升到5级时，他马上开始受到-20%的XP减值，直到两个职业的等级差回到平衡为止。

种族与兼职经验值：种族的适合职业(见第二章)不计入XP减值。此时，该人物视为不具有该职业。举例而言，柏格温是11级侏儒(9级游荡者/2级吟游诗人)，他不会受到XP减值，因为他只有一个职业不适合其种族(吟游诗人是侏儒的适合职业)。如果他升到12级(人物等级)并决定要再兼职战士，成为9级游荡者/2级吟游诗人/1级战士。此后，他必须受到-20%的XP减值，因为他的战士等级远低于游荡者等级。假设下次冒险本来可以得到1200点XP，他只能得到80%，也就是960点。如果他升到13级(人物等级)，并且又再兼职牧师，则他必须开始受到-40% 的XP减值。

又比如说：7级战士/2级牧师的矮人不会受到XP减值，因为战士是矮人的适合职业，所以不列入计算。即使升级时他再兼职1级游荡者，也不会受到XP减值，因为牧师与游荡者的等级只差1级。

人类或半精灵兼职时，等级最高的职业视为其适合职业。

兼职的方法

4级半身人游荡者莉达决定要学习法师法术，增加自己的才能。她找到一个法师当导师，而且仔细观察队伍里的法师同伴米雅莉如何准备法术。当莉达得到10000点经验值时，她可以升到5级。然而，她决定不成为5级游荡者，而升为4级游荡者/1级法师。现在，她得到1级法师的好处，但牺牲了成为5级游荡者的优点。她可以得到一个法师生命骰数(d4)、1级法师的意志检定+2加值，以及4点技能点数(2点来自法师职业，加上她智力14的+2加值)。这些数值累计到她原本身为游荡者的数值上。她的基本攻击加值、反射检定和强韧检定并未增加，因为1级法师的这些数值是+0。她得到1级法师的起始法术书，以及每日可施展的法术数量，然而她的游荡者技能和偷袭能力并未增加。她可以把她得到的4点技能点数，分一部分给游荡者技能，但此时这些游荡者技能将视为跨职技能，所以1点技能点数只能换半个技能级数。("工艺"或"专业"技能除外，因为这两者同时为法师和游荡者的本职技能)。

得到15000点经验值时，莉达成为6级人物。她决定继续走法师之路，再度提升法师等级而非游荡者等级。如此一来，她可再次获得法师(而非游荡者)升级的好处。身为2级法师，她再次得到一个d4生命骰数，她的基本攻击加值和意志检定加值再+1，她还得到4点技能点数，并且每天可以多准备一个零级法术，以及一个一级法术(见表3-18：法师)。此外，由于她的人物等级为6级，她可以获得第三个专长(见22页表3-2：经验值与升级影响)。

此时，莉达的人物等级是6级，职业等级则分别是4级游荡者/2级法师。她可以如同2级法师一样施法，也可以如4级游荡者般进行偷袭。她的战斗技巧比4级游荡者略胜一筹，因为她在学习法术时也锻炼了些微战斗技巧(在法师等级升上2级时，基本攻击加值+1)。她的基本反射检定加值为+4(游荡者职业+4，法师职业+0)，比6级法师好，但不如6级游荡者。她的基本意志检定加值为+4(游荡者职业+1，法师职业+3)，比6级游荡者好，但不如6级法师。

每次升级时，莉达可以决定要提升游荡者等级或法师等级。当然，如果她还想要得到更广泛的能力，她甚至可以选择第三个职业，比方说战士。
