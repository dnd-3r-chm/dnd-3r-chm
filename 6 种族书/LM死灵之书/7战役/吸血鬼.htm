吸血鬼变体(Vampire Variants )

      

恐月吸血鬼(Moonbane Vampire )

恐月吸血鬼惧怕满月的光。

当恐月吸血鬼暴露在满月的月光下。会变得迟缓(DC15意志豁免。通过则不受影响)。若月亮不是满月。吸血鬼仍然会受影响。但豁免有+4加值。没有月亮的夜晚。吸血鬼不受影响。(想象艾伯伦的情况……12个月亮……汗……)

CR调整:+0.

      

迷诱吸血鬼(Persuasive Vampire )

迷诱吸血鬼使用它的力量和语言来动摇倾听者的意志

特殊攻击:

暗示(Suggestion。 Su ):迷诱吸血鬼可以对30漳谌我馐量的目标进行暗示(意志豁免。通过则无效)。这等效于暗示术(suggestion)。CL12。吸血鬼可以选择对那些目标使用此能力。迷诱吸血鬼失去支配特殊攻击。

CR调整:+0

      

心灵吸血鬼(Psychic Vampire )

心灵吸血鬼吸取对手的心灵强度，而非物理生命。

特殊攻击:

感知吸取(Wisdom Drain。 Su ):只要成功进行接触攻击或挥击。心灵吸血鬼从目标处吸取1d6点感知。同时获得5点临时生命。持续1小时。每轮可以如此吸取1次。另外。吸血鬼身边10漳诘幕钗锩柯侄蓟岜晃血鬼吸取1点感知(意志豁免。DC=10+1/2HD+魅力调整。通过则无效)。心灵吸血鬼不会从此能力获得任何临时生命。心灵吸血鬼可以以标准动作激活或压制它的感知吸取灵光。

心灵吸血鬼不获得能量吸取能力。

CR调整 :+0

     

荒野吸血鬼(Savage Vampire )

攻击:荒野吸血鬼保留基础生物的所有攻击方式。并获得两爪击。若它还没有此种攻击方式。(没有手的生物仍然如同普通吸血鬼获得挥击)

伤害:荒野吸血鬼的爪击造成与同体型普通吸血鬼的挥击同样的伤害。

特殊攻击:荒野吸血鬼失去支配(dominate)特殊攻击。若荒野吸血鬼获得爪击。每个爪子命中都会造成一个负向等级。

属性:在基础生物之上增加:力量+10。 敏捷+4。 智力-2。 感知+2. 作为不死生物。荒野吸血鬼没有体质。

技能:荒野吸血鬼在躲藏。跳跃。 聆听。潜行和侦察检定上有+8种族加值。

CR调整:+0

 

集群形态吸血鬼(Swarmform Vampire )

集群形态吸血鬼可以变成生物集群的形态。

特殊能力:取代普通吸血鬼变形能力中列举的形态。集群形态吸血鬼可以变形为老鼠或蝙蝠集群。

CR调整:+0
