魂能之刃 Incarnum Blade

“万灵英魂指引我的剑刃。”――永恒战士的领袖，魂能之刃维克托兰・马林男爵

通过世代相传之秘，魂能之刃从历代最伟大战士身上汲取灵魂能量塑造成一种特殊灵魂融合，并将其融入自身所选的近战武器中，剑魂将赋予他惊人的战斗力。当剑魂与脉轮连接，它会为使用者灌注魂能的力量，增幅他的战斗能力。待魂能之刃死后，他将与剑魂中的灵魂能量融合，成为指引日后使用者无数先灵的一部分。

成为魂能之刃：由于魂能之刃本质上是个战士，所以最简单、也是最明了的进阶途径就是以战士、圣武士之类的战斗职业进阶，以尽早满足基本攻击加值的进阶条件。在战斗向的职业中，本职技能包括专注的（如天魂者、圣武士和游侠）职业在技能上比战士占优，只不过这些角色若想追寻魂能之刃的道路就得放弃自己的灵魂融合或施法能力的提升。不过由于本进阶职业所带来的能力适合任何专注近身战斗的角色，所以游荡者、武僧和斥候（见完美冒险）也有可能会为永恒战士高举刀剑。

且由于魂能之刃是一个只有5级的进阶职业，因此它能让角色获得新的独特能力，又不会让自己的基础职业或其他进阶职业彻底走偏。

职业能力

该进阶职业的基本获益为剑魂合一以及该灵魂融合链接到脉轮的效果。随着你提升魂能之刃的等级，你将获得剑魂合一进一步的应用。最终你能迅速转移脉轮链接和同时将剑魂合一链接到两个脉轮之上。

 

剑魂合一（Su）：1级开始，你获得创造剑魂合一，一种特殊灵魂融合的能力。剑魂合一将魂能包缠到选定的近战武器上，使得该武器更加接近你的灵魂和意念，并给予它特殊的能力。带有剑魂合一的武器呈现出蓝色的光芒，并视为魔法武器因此能击破相应的伤害减免。该武器的硬度增加5点，生命值增加5点/魂能之刃等级，并在豁免上获得等同于你魂能之刃等级的加值。

塑造剑魂合一如同普通的灵魂融合一样，需要8小时休息和1小时冥想。剑魂合一持续存在，除非你自行解消，或是融合的武器被击破或以任何方式破坏。仅仅松手放开武器不会导致剑魂合一解消，即使你不握持，该武器硬度、生命值和豁免加值都会保留。持有者通过剑魂合一的获益并不会对任何其他捡起来的人。

 

剑魂合一脉轮链接（Su）：当你塑造剑魂合一的时候，你能将它链接到你身体的一个脉轮之上。只要剑魂合一存在，并且你握持该武器，你便能获得相应的特殊能力。与其他灵魂融合不同，链接到脉轮并不会改变该灵魂融合的外观，被影响的脉轮外表也不会有任何变化。而且，被剑魂合一缠绕的武器依然能正常佩戴，即使它不是被链接到你的手掌。你能被正常地卸除武器，也能被击破武器。如果你在灵魂融合链接到脉轮的情况下松手或放下武器，它依然是链接到你的脉轮，同时依然有着剑魂合一的特性，你也依然拥有脉轮链接的特殊能力。其他任何人都无法享受到该武器基于魂能的能力。

与其他的脉轮链接不同，剑魂合一不会占用你的装备栏。例如，链接到手掌的剑魂合一不会妨碍你穿戴魔法手套。剑魂合一脉轮链接不会给你将其他灵魂融合或魔法物品链接到脉轮的能力。

1级起，你能将剑魂合一链接到头冠、脚或手掌脉轮上，2级增加了手臂、额头和肩膀脉轮，3级增加了喉咙和腰，4级增加了心，5级增加了灵魂。剑魂合一脉轮链接的能力如下：

n 头冠脉轮：只要你链接头冠脉轮，对手夹击你就不会获得正常+2加值，尽管由于夹击而产生的其他效果还在，例如对方依然可以偷袭你。你在借机攻击的攻击骰和避免卸除武器的检定获得+4洞察加值，同时你在对抗战斗中的虚招中的察言观色检定中获得+4洞察加值。

n 脚脉轮：只要你链接脚脉轮，你在先攻检定中获得+2洞察加值，除此之外，即使处于措手不及的状态，你依然能用剑魂合一的武器进行借机攻击。

n 手掌脉轮：你在使用剑魂合一武器攻击时，在攻击骰和伤害骰上获得+1洞察加值。

n 手臂脉轮：你在使用剑魂合一武器确认重击的攻击骰中获得+4洞察加值。

n 额头脉轮：若你使用剑魂合一武器攻击拥有隐蔽的目标时失手，你能重投一次失手几率以确定是否命中，该能力类似于盲斗专长。如果你已经拥有盲斗专长了，则该能力不会有任何效果。也就是你只能重投1次失手几率。

n 肩膀脉轮：你有25%几率无视重击的额外伤害。你不需要意识到该次攻击，但是在你失去意识或无助的状况下无效。该能力无法无视偷袭伤害骰，且不能和盔甲或盾牌的特殊效果叠加。

n 喉咙脉轮：你能挥动武器并发出一阵战吼。以一个标准动作，你能让60尺范围内听到你的敌人战栗1轮（通过意志豁免可以抵御该效果，DC=10+魂能之刃等级+体质调整值）。你能随意使用该能力。

n 腰脉轮：你在对抗冲撞、擒抱、摔绊或闯越的对抗检定上获得+10洞察加值。你即使处于措手不及状态时也能保佑AC上的敏捷加值。

n 心脉轮：你获得等同于你角色等级2倍的生命值（最大+40），该额外生命值不像临时生命值一样最先失去，如果你改变剑魂合一的脉轮链接，你马上失去该生命值。

n 灵魂脉轮：你的剑魂合一武器获得阵营效果以克服相应的伤害减免。当你链接该脉轮时，你可以选择任意一个符合你阵营的阵营构成（混乱、邪恶、善良、守序）。之后无法改变除非你重新塑造剑魂合一。对于你所选择的阵营构成的对立阵营亚种，你的武器造成额外1d6点伤害。如果剑魂合一武器本来已经拥有阵营武器特性，则额外伤害可以叠加。

 

重链剑魂合一（Su）：3级起，你获得改变剑魂合一所链接脉轮的能力。你每天能使用该能力1+体质加值的次数（最少1次）。重链剑魂合一是个不会引起借机攻击的标准动作。

5级时，你能以一个标准动作同时改变剑魂合一所链接的两个脉轮。

 

双重脉轮：到达5级后，你能同时将剑魂合一链接到两个脉轮之上，第二个脉轮必须比第一个脉轮低1级。例如，如果你第一个脉轮是喉咙脉轮（魂能之刃3级），则第二个脉轮不能是腰脉轮（魂能之刃3级），而只能是头冠、手掌、脚（魂能之刃1级）或额头、手臂、肩膀（魂能之刃2级）。你从两个链接中都能获得特殊能力，如同你正常分别使用该链接。

 

魂能之刃逸闻

拥有逸闻知识或在知识（历史）上有等级的角色可能会知道某些历史上的英雄是魂能之刃，或者名为永恒战士的组织。知道这些情报的角色能进一步地了解研究魂能之刃及其组织。通过相应DC的逸闻知识或知识（历史）检定能获得的情报如下：

DC10：虽然大部分战士拥有强力的魔法武器，但是有少部分战士带有缠绕着与别不同能量的武器。

DC15：名为魂能之刃的战士能用奇怪的灵魂力量强化他们的武器。大部分都是属于名为永恒战士的组织，一个精英佣兵团体。

DC20：魂能之刃曾经宣誓在死后要将全心全灵奉献与给予下一代永恒战士以力量。

DC30：Baron? Malin，永恒战士的领导者，一直在收集最尊贵最勇敢的英雄的灵魂。他的目的尚算正当，然而英雄们却因此牺牲而奉献出他们的灵魂。

在永恒战士所经营的聚居点或防御工事附近，一个DC20的收集信息检定能让玩家顺利与该佣兵组织接触，如果玩家想要雇佣佣兵或被录用的话。
